# Lista de Jogos

O projeto consiste na criação de APIs para o retorno de diferentes listas de jogos como:
Ex: 
- Jogos de RPG
- Jogos de Esportes
- Jogos de Aventuras e ação

Para isso foi utilizado as seguintes tecnologias
- Java
- Spring Boot
- H2
- Hibernate
- Json
- Postgreen
